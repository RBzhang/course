## 数据说明，以训练数据为例
```python
文件名即为标签,即1.npy代表第一类数据；2.npy代表第二类数据；以此类推。
针对每类数据：
训练数据维度为 (5980,2,5)；表示包含5980个样本，2个通道，每个通道包含5个特征； 
测试数据维度为 (2530,2,5)；表示包含2530个样本，2个通道，每个通道包含5个特征；  
所以总的训练 41860=5980*7: 测试数据 17710=2530*7;

训练数据维度（5980，2，5）；包含两个通道a,b，分别对应 索引 0和1；
import numpy as np
data = np.load('1.npy') # (5980,2,5)
a = data[:,0,:] # (5980,1,5)
b = data[:,1,:] # (5980,1,5)
需要三种实验方案：
1.使用全通道数据训练，测试全通道数据
2.仅使用a通道数据训练，测试a通道数据
3.仅使用b通道数据训练，测试b通道数据
```
实验方案|输入数据的维度|模型
--------|--------|--------
 1 | 2*5 |模型1
 2 | 1*5 |模型2
 3 | 1*5 |模型3
